# SOUGOU STORE 動作確認手順書

本手順書は、開発チーム外の方がリポジトリをクローンし、アプリケーションの動作を確認するためのものです。

---



## セットアップ手順

### Step 1：リポジトリをクローンする

```bash
git clone https://github.com/Yagoshi/sougou-enshu.git
cd sougou-enshu
```

---

### Step 2：パッケージをインストールする

```bash
pip install -r requirements.txt
```

---

### Step 3：.env ファイルを作成する

`.env.example` をコピーして `.env` を作成します。

```bash
cp .env.example .env
```

`.env` をテキストエディタで開き、以下の項目を自分の環境に合わせて編集してください。

```
DJANGO_SECRET_KEY=任意の長いランダム文字列
DJANGO_DEBUG=True
DJANGO_ALLOWED_HOSTS=localhost,127.0.0.1

DB_NAME=sougou-enshu        # 後述のStep 4で作成するDB名
DB_USER=root                # MySQLのユーザー名
DB_PASSWORD=P@ssw0rd        # 自分のMySQLパスワードに変更
DB_HOST=127.0.0.1
DB_PORT=3306

API_KEY=your-api-key-here   # Step 9で取得（決済機能の確認に必要）
GEMINI_API_KEY=             # 任意（AIチャットボット機能の確認に必要）
```

> ⚠️ `.env` はGit管理対象外です。絶対にコミットしないでください。

---

### Step 4：MySQLでデータベースを作成する

```bash
mysql -u root -p
```

パスワードを入力してログインしたら、以下のSQLを実行してください。

```sql
CREATE DATABASE `sougou-enshu`;
exit;
```

> ⚠️ DB名にハイフン（`-`）が含まれるため、バッククォートで囲む必要があります。

---

### Step 5：マイグレーションを実行する

```bash
python manage.py migrate
```

エラーなく完了すればOKです。

---

### Step 6：商品・カテゴリデータを登録する

```bash
python register_items_with_images.py
```

以下のように表示されればOKです。

```
Step 1: カテゴリ登録
  [1] 新規作成: 鞄
  ...（中略）
Step 2: 既存商品をクリア
Step 3: 画像付き商品を登録
  [200] ✅ レザートートバッグ Mサイズ  📷 item_200_...png
  ...（中略）
✅ 登録完了: 100件
```

> ⚠️ 商品画像（`media/items/`）はGit管理対象外のため、別途配布された `media_items.zip` を使用してください。
> 解凍方法は以下のとおりです。
>
> ```bash
> unzip media_items.zip
> ```
>
> 画像がない場合でもアプリは動作しますが、商品画像は表示されません。

---

### Step 7：会員・注文・評価データを登録する

```bash
python manage.py seed --clear
```

以下のように表示されればOKです。

```
会員: 5件
カート内商品: 5件
注文: 5件
注文詳細: 5件
管理者: 5件
評価データ: 500件（100商品 × 5件）
=== サンプルデータの投入が完了しました ===

ログイン情報:
  会員: user001 / password001 〜 user005 / password005
  管理者: admin001 / adminpass001 〜 admin005 / adminpass005
```

---

### Step 8：開発サーバーを起動する

```bash
python manage.py runserver
```

ブラウザで http://127.0.0.1:8000 を開いてトップページが表示されればセットアップ完了です。

> 404確認ページは http://127.0.0.1:8000/404/ で確認できます（`DJANGO_DEBUG=True` 時のみ）。

---

### Step 9：決済APIキーを取得する（支払機能の確認に必要）

1. ブラウザで http://15.152.44.182/ を開く
2. 「新規登録はこちら」からアカウントを作成する
3. ダッシュボードに表示されるAPIキーをコピーする
4. `.env` の `API_KEY=` に貼り付けて保存する
5. サーバーを再起動する（Ctrl+C → `python manage.py runserver`）

### ① トップページ・商品一覧

| 手順 | 期待する結果 |
|---|---|
| http://127.0.0.1:8000 を開く | 商品一覧が表示される |

## エラーが出た場合

**`Unknown database 'sougou-enshu'`**

MySQLにデータベースが作成されていません。Step 4 を実行してください。

```sql
CREATE DATABASE `sougou-enshu`;
```

---

**`No module named 'dotenv'`**

```bash
pip install python-dotenv
```

---

**`No module named 'pymysql'`**

```bash
pip install pymysql
```

---

**`No module named 'PIL'`**

```bash
pip install Pillow
```

---

**商品画像が表示されない**

`media/items/` フォルダに画像ファイルがありません。配布された `media_items.zip` を `sougou-enshu/` 直下に置いて解凍してください。

```bash
unzip media_items.zip
```

---

**ログインできない**

サンプルデータが正しく投入されていない可能性があります。Step 6・Step 7 を再実行してください。

```bash
python register_items_with_images.py
python manage.py seed --clear
```

---

**評価データが表示されない**

商品データの後に seed を実行する必要があります。以下の順番で実行してください。

```bash
python register_items_with_images.py   # 先に商品登録
python manage.py seed --clear          # 次に評価データ登録
```
