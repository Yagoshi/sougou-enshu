# 動作確認手順書

## 事前準備

### 1. 最新のdevelopを取得する

```bash
git checkout develop
git pull origin develop
```

### 2. 必要なパッケージをインストールする

```bash
pip install -r requirements.txt
```

### 3. .envファイルを作成する

`.env.example` をコピーして `.env` を作成してください。

```bash
cp .env.example .env
```

VS Codeで `.env` を開いて、以下の項目を自分の環境に合わせて編集してください。

```
DB_NAME=sougou-enshu       # 自分のDBの名前に変更
DB_PASSWORD=P@ssw0rd       # 自分のMySQLパスワードに変更
API_KEY=your-api-key-here  # 自分のAPIキーに変更（取得方法は後述）
```

### 4. MySQLでデータベースを作成する

```bash
mysql -u root -p
```

```sql
CREATE DATABASE `sougou-enshu`;
exit;
```

### 5. マイグレーションを実行する

```bash
python manage.py migrate
```

### 6. サンプルデータを投入する

```bash
python register_items_with_images.py
```

### 7. APIキーを取得する（支払API動作確認用）

1. ブラウザで **http://15.152.44.182/** を開く
2. 「新規登録はこちら」をクリックしてアカウントを作成する
3. ダッシュボードに表示されるAPIキーをコピーする
4. `.env` の `API_KEY=` の値に貼り付けて保存する

### 8. 開発サーバーを起動する

```bash
python manage.py runserver
```

ブラウザで **http://127.0.0.1:8000** を開いてください。

## エラーが出た場合

### よくあるエラーと対処法

**`Unknown database 'sougou-enshu'`**
→ MySQLでDBを作成してください。
```sql
CREATE DATABASE `sougou-enshu`;
```

**`No module named 'dotenv'`**
→ パッケージをインストールしてください。
```bash
pip install python-dotenv
```

**`No module named 'pymysql'`**
→ パッケージをインストールしてください。
```bash
pip install pymysql
```

**ログインできない**
→ seedデータを再投入してください。
```bash
python manage.py seed --clear
```

**その他のエラー**
→ エラーメッセージをSlackまたはPMに共有してください。
